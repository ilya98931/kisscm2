##Конфигурационное управление. Этап 1 Минимальное CLI-приложение с конфигурацией
Настраивыемые параметры:
-Имя анализируемого пакета.
-URL-адрес репозитория или путь к файлу тестового репозитория.
-Режим работы с тестовым репозиторием.
-Версия пакета.
Для этого этапа при запуске приложения вывести все параметры, в формате ключ значение
Реализовать и продемонстрировать обработку ошибок для всех параметров.
##Предварительное требования
-Python 3.7
-git
##Как скачать и запустить 
git clone https://github.com/ilya98931/kisscm2.git
